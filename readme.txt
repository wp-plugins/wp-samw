WordPress - Sidebar Accordion Menu Widget

== Description ==
**W** ord**P**ress - **S**idebar (Simple) **A**ccordion **M**enu **W**idget allows to add a sidebar menu as quick and easy as WordPress default Custom Menu.
Difference: Submenues will be hidden and displayed with a simple accordion effect.

No need of cookies, flash or other additional requirements, no complicated adjustments. 

== Installation ==
1. Upload *wp-samw* folder to the */wp-content/plugins/* directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add *SAMW Custom Accordion Menu* to your widgetable area and configure the options.
